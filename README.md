
# NAME

WWW::GetMetaApi - Gets the meta data on other web sites, and can be called from \*.user.js

# SYNOPSIS

    use WWW::GetMetaApi;

# DESCRIPTION

WWW::GetMetaApi is an api to be used from some user scripts, and return the meta data of other web sites.
